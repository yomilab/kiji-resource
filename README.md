# KiJi Resource

Recommended RSS feeds for KiJi. Browse by topic or import a bundle into the app.

**Quick start:** In KiJi, use **Import OPML** and paste a link from the **Import** column below. Start with [Recommended](https://raw.githubusercontent.com/yomilab/kiji-resource/main/recommended.opml) to see every category.
